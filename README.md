# ML Experiments

Hands-on implementations of core machine learning algorithms from scratch,
organized by topic. Built while learning the fundamentals behind each model
before applying them to real problems.

## Topics Covered

| Topic | Algorithm | Notes |
|-------|-----------|-------|
| `linear-regression` | Linear Regression | OLS, gradient descent |
| `polynomial-regression` | Polynomial Regression | Feature expansion, overfitting |
| `logistic-regression` | Logistic Regression | Binary classification, sigmoid |
| `decision-tree` | Decision Tree | Entropy, information gain |
| `random-forest` | Random Forest | Ensemble, bagging |
| `svm-linear` | Support Vector Machine (Linear) | Hyperplane, margin maximization |
| `svm-nonlinear` | Support Vector Machine (Non-linear) | Kernel trick, RBF |
| `pca-dimensionality-reduction` | PCA | Eigenvectors, variance explained |
| `data-preprocessing` | Preprocessing Pipelines | Scaling, encoding, imputation |

## Stack
Python · scikit-learn · NumPy · Pandas · Matplotlib · Jupyter
